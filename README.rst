Contents
========

.. contents::


Install
=======

- Add eea.package to your zopeskel eggs section within your buildout and
  re-run buildout.

Getting started
===============

- Run 'zopeskel eea' to create new eea package

Source code
===========

- Latest source code (Plone 4 compatible):
